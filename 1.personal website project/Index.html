Space Syntax and its Theory

Space syntax is a typical tool integrated with the typological model with the by examing the space.The linear space can be abstracted as intersected lines so that can be analyzed with the topology. By linking to the humans' activity, some basic patterns can be found.

This tool plays key impact on my future career. It is from that time I have realized I should learn more about the computer and web to get to know the basic theories behind these tools.
